# CLEANTHI Alpha-olenic Oil LTD

Static one-page website for a premium high-polyphenol extra virgin olive oil brand from Cyprus.

## Files

- `index.html` - Website markup
- `styles.css` - Website styling
- `assets/cleanthi-hero.png` - Hero/product image
- `assets/cleanthi-logo.png` - Brand logo

## GitHub Pages

This site does not need a backend or build step. To publish it with GitHub Pages:

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Choose the `main` branch and `/root` folder.
5. Save.

GitHub will provide a public website URL after the page is deployed.
